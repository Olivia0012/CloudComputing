# percent foreigner

## how to run
```python main.py```
- if you add more json file, just add the file name in ```data_file.json``` and rerun the python script and result file will be ready

## format
### percent foreigner data
```json
{
    "lga_code": {
        "percentage_foreigner": float,
    },
}

```

#### all "lga_code"
```
["Greater Adelaide"
"Greater Melbourne"
"Greater Brisbane"
"Greater Sydney"]
```
